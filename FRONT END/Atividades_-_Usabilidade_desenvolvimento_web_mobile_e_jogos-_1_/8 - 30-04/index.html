Modelo de Caixa

O modelo de caixa CSS é essencialmente uma caixa que envolve cada elemento HTML. Consiste em: bordas, preenchimentos, margens e o conteúdo real.

Este texto é o conteudo da caixa
Adicionamos ume preenchimento de 50px, uma margem de 20px e uma borda verde de 15px. Pois, para chegar ao mais infinito detalhe, ninguém deve praticar qualquer tipo de trabalho, a menos que tire algum proveito disso.
